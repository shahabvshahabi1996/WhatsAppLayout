# whatsAppLayout

> WhatsApp Layout Using React-Native

## Installation

```sh
$ npm install -g react-native-cli
```
```sh
$ clone this project
```
```sh
$ npm || yarn install 
```
```sh
$ react-native run-android 
```

## Usage

```js
var whatsapplayout = require('whatsAppLayout');
whatsapplayout();
```
